# Maverick Landing Page

A polished React/Vite landing page for Maverick, an early AI-native hardware development platform.

## Run Locally

~~~bash
npm install
npm run dev
~~~

Then open the local Vite URL shown in the terminal.

## Build

~~~bash
npm run build
~~~

The site is intentionally positioned as a working prototype / early product. It avoids fake traction, fake partnerships, and production-readiness claims.
